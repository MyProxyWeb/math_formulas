# Typst Math Project

A minimal Typst skeleton for math formulas. Edit `main.typ` or split into `sections/*.typ`.

Build / preview:

- `typst preview main.typ` # live preview (if you have the Typst CLI or editor plugin)
- `typst compile main.typ out.pdf` # produce a PDF

You can also view the compiled PDF here:  
[main.pdf](https://github.com/MyProxyWeb/math_formulas/blob/main/out.pdf)
